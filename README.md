# TicketsApp 🎟️

**TicketsApp** is a platform for browsing and purchasing tickets to a wide variety of events. Users can explore events by categories, while admins have full control over event management.

## Features ✨

## 🔐 Authentication
- Users must log in to access the Events and Categories pages

## 🎫 Event Management
- Admins can **create**, **edit**, and **delete** events
- Manage event details: **title**, **description**, **date**, **time**, **location**, **price**, and **category**
- Upload and display **event images**
- Event cards are shown in a responsive pastel-themed grid layout

## 🗂️ Category Management
- Admins can **create**, **edit**, **delete**, and **view** event categories
- Events are assigned to categories like **Festival**, **Concert**, **Art Show**, **Sports**, **Esports**

## 👨‍💼 Logged-In UI
- Navigation bar changes depending on login status:
  - If **not logged in**: only **Home** is visible
  - If **logged in**: **Events**, **Categories**, and **Logout** links appear
