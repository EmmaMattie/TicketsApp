# TicketsApp 🎟️

**TicketsApp** is a platform for browsing and purchasing tickets to a wide variety of events. Users can explore events by categories, while admins have full control over event management.

## Features ✨

- **Event Management**:
  - Admins can **create**, **edit**, and **delete** events.
  - Manage event details such as **title**, **date**, **location**, **price**, and **category**.
  - Admins can upload and manage **event images** for better visual representation.
  
- **Event Categories**:
  - Events are organized into **categories** such as music, festivals, concerts, sports, and arts.
  - Users can explore events within specific categories.

- **User Interaction**:
  - Users can view detailed information about each event, including images and pricing.
  - **Details page** for each event with more information.
  
- **Admin Dashboard**:
  - Easy-to-use interface for managing events and viewing all existing events.
  - Admins can access a list of all events with options to **edit** or **delete** them.
