# TicketsApp 🎟️

**TicketsApp** is a pastel-themed ticket sales platform where users can browse and purchase tickets for concerts, festivals, art shows, sports, and more. Admins manage event listings, while visitors enjoy a smooth shopping experience.

## 🔐 Authentication
- Users must log in to access the Events and Categories pages

## 🎫 Event Management
- Admins can **create**, **edit**, and **delete** events
- Manage event details: **title**, **description**, **date**, **time**, **location**, **price**, and **category**
- Upload and display **event images**
- Event cards are shown in a responsive pastel-themed grid layout

## 🗂️ Category Management
- Admins can **create**, **edit**, **delete**, and **view** event categories
- Events are assigned to categories like **Festival**, **Concert**, **Art Show**, **Sports**, **Esports**

## 🛍️ Shopping Cart & Checkout
- Visitors can **browse public events** on the Browse page
- Add tickets to a **shopping cart**
- View a **cart summary** with event name, quantity, and total
- Complete a **simulated checkout** (no payment required)
- See a **confirmation page** after checking out

## 🖼️ User Interface
- Pastel color palette throughout the site
- Clean and responsive layout
- Dynamic navbar updates based on login status:
  - If **not logged in**: only **Home** and **Browse** are visible
  - If **logged in**: **Events**, **Categories**, **Cart**, and **Logout** appear
